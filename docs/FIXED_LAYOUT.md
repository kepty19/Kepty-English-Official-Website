# 固定 PC レイアウト（全端末共通）

サイト全体は **幅 1440px の PC レイアウト** で固定しています。

## 仕組み

| 設定 | 役割 |
|------|------|
| `index.html` の `viewport width=1440` | スマホ・タブレットでも 1440px として描画し、画面に収まるよう縮小表示 |
| `#root { width: 1440px }` | ワイドモニターでもレイアウトが伸びず、中央に同じ UI |
| `body { min-width: 1440px }` | PC ブラウザを狭くしたときは横スクロール（レイアウトは変えない） |

## 端末ごとの見え方

- **スマホ / iPad** … PC と同じ配置・同じ文字サイズ比で縮小表示（ピンチで拡大可能）
- **ノート PC（≥1440px）** … そのまま表示
- **ワイドモニター** … 1440px の UI が中央、左右はヒーロー色の余白

## 注意

- スマホでは文字が小さく見えるため、必要ならピンチズームしてください。
- レスポンシブ用の `sm:` / `md:` は「1440px 幅の PC」として常に有効な側が選ばれます。

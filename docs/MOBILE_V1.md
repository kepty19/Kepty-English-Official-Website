# スマホ UI v1

方針 **A**：PC と同じ情報量を、スマホ向けの配置・サイズで表示。

## v1 で入れたもの

| 領域 | スマホ | PC（md 以上） |
|------|--------|----------------|
| ナビ | ハンバーガーメニュー（右上） | 従来のピル型タブ |
| ヒーロー高さ | `100dvh` | `h-screen` |
| ロゴ | 小さめ・ハンバーガー下 | 従来サイズ |
| 会社名 | 非表示 | 右上表示 |
| タイトル | 左寄せ・折り返し可・小さめ | 大見出し・1行 |
| 説明文 | 17px・下余白調整 | 33px |
| 3D ビジュアル | 中央下・小さめ | 右下・大きめ |

## ブレークポイント

- **`< 768px`** … スマホ UI（Tailwind デフォルト）
- **`md:` (≥768px)** … PC UI

## フィードバックの書き方

```
HOME
- ヒーロー画像：もう少し右
- タイトル：1行にしたい

SERVICE
- 表：横スクロールでOK
```

セクション名 + 具体的な希望で v2 に反映します。

## 固定 1440px レイアウトについて

v1 では **解除済み**（`viewport: device-width`、可変幅）。
